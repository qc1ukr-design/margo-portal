## Способи оплати (payment_type у чеку)
| Тип | Де зустрічається | Де шукаємо в банку |
|---|---|---|
| NovaPay інтегратор | Доставка НП | NovaPay реєстр → NovaPay банк |
| Готівка | Магазин, кур'єр | Не шукаємо (MATCHED_CASH) |
| Термінал (картка) | Магазин | Еквайринг у виписці банку |
| LiqPay | Онлайн | LiqPay API → банк |
| IBAN / р-р | B2B, передоплата | Пряма виписка банку |
| ФОП → ФОП | Внутрішні переміщення | Виписка банку, особливий тип |
| Rozetka Pay | Маркетплейс | ФК ЕВО → банк |
| Prom.UA | Маркетплейс | Prom API → банк |
